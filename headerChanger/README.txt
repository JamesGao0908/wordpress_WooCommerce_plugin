This plugin is based on WooCommerce. 
In product page, it will remove the footer bar and insert three new button, 'home', 'buy_now' and 'add_into_cart'. 
In cart page, it will remove the 'checkout' button from the screen into footer bar.
You can see these changes only in a small screen for example phones.

(这是基于WooCommerce的一个插件。在小尺寸(手机屏幕)时，产品界面的footer会被移除。
并且加入三个新的按钮，分别为“店铺”、“立即购买”和“加入购物车”。
购物车界面的结算按钮会被移到底部。)
